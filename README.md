# מתמטיקה לפיזיקאים II — ארכיון בחינות

ארכיון של בחינות הקורס **מתמטיקה לפיזיקאים II** (86-148, סמסטר ב') של פרופ' ליאור קליין באוניברסיטת בר-אילן, עם פתרונות מלאים.

🔗 **לאתר**: <https://lk-physics.github.io/math-phys-2-exams/>

## מה יש באתר?

- שאלות מבחינות שנים 1999–2025 (כ-49 בחינות בסך הכל)
- פתרונות מפורטים לכל שאלה, עם הסברים וצעדי החישוב
- חיפוש לפי מילה בניסוח השאלה ובגוף הפתרון
- בורר נוח לבחינה לפי שנה ומועד
- קישור ל-PDF המקורי של כל בחינה

## למי האתר מיועד?

לסטודנטים בקורס המתכוננים לבחינה. הפתרונות נועדו לתרגול עצמי — לא רק לקבלת התשובה אלא להבנת השיטה.

## הערה

הפתרונות נכתבו על ידי פרופ' קליין במסגרת פרויקט הוראה. אם מצאתם טעות או הצעה לשיפור, אנא דווחו דרך [Issues](https://github.com/LK-physics/math-phys-2-exams/issues).
